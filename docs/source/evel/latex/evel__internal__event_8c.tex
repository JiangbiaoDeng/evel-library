\subsection{Detailed Description}
Implementation of E\+V\+E\+L functions relating to the internal events. 

Internal events are never expected to be sent to the J\+S\+O\+N A\+P\+I but comply with interfaces for regular event types. The primary use-\/case is to enable the foreground processing to communicate with the background event handling processing in an orderly fashion. At present the only use is to initiate an orderly shutdown of the Event Handler thread.
